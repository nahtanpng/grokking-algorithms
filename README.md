# Repositório com códigos de exemplo e exercícios do livro "Grokking Algorithms" de Aditya Bhargava.

Este repositório contém códigos de exemplo e exercícios do livro "Grokking Algorithms" de Aditya Bhargava. O livro é uma introdução amigável e visual aos algoritmos, projetada para ajudar os leitores a entenderem os conceitos fundamentais de algoritmos de forma clara e acessível.
